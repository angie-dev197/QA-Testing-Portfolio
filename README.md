## QA Project — Demo Website (Manual + Automation)

A portfolio project demonstrating an end-to-end QA workflow on a self-built 
multi-page static website (Home, Services, Culture, Contact).

**Scope**
- Structured + exploratory manual testing across 4 browsers/devices
- Automation suite built with Python + Selenium + pytest (navigation, form validation, responsive breakpoints)
- Risk-based test prioritization (P1/P2), defect reporting with evidence, and retest cycles

**Tech stack:** Python · Selenium · pytest · HTML/CSS/JS · Git · VS Code

**Deliverables:** Test plan · Test cases · Bug reports with screenshots · Execution summary · Automation scripts

> Designed around real constraints: solo tester, limited hardware, free tooling only —
> coverage decisions were intentional and documented.

*Full test plan is available in the [repository files](./QA-Manual-Testing/test_plan.md).*
